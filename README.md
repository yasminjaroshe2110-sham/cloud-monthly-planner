# Cloud Monthly Planner

A cloud-based monthly planner application developed as a final project for the Cloud Development course.

## Features

- Monthly calendar view
- Add tasks with date and time
- Task categories
- Edit tasks
- Delete tasks
- Mark tasks as completed
- Upcoming tasks dashboard
- Statistics dashboard
- Local storage persistence

## Technologies

- HTML
- CSS
- JavaScript
- AWS
- GitHub
- GitHub Actions
- CloudFront
- Route53
